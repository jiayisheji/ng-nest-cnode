# ng-nest-cnode

Angular 8 Front-End and Nestjs framework Back-End build Fullstack CNode

## Quick Start

打开命令行工具：

```bash
# 克隆项目
git clone https://github.com/jiayisheji/ng-nest-cnode.git
# 进入项目工作区
cd ng-nest-cnode
# 安装依赖
npm install
# 打开vs code
code .
```

## Development server

运行客户端

```bash
npm start client
```

运行服务端

```bash
npm start server
```
